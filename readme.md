# 픽셀 서바이벌

## 1. 게임 컨셉
   
   뱀파이어 서바이벌 느낌의 2D 로그라이크 서바이벌 게임
   
## 3. 핵심 요소

   ### ㄱ. 자동 공격

   ### ㄴ. 몬스터의 추적 및 경험치, 레벨업 시스템

   ### ㄷ. 장비와 세트 효과

## 4. 개발 범위

   ### ㄱ. 플레이어 - 8방향 이동, 하단 체력바
   
   ### ㄴ. 몬스터 - 일반 몬스터(플레이어 추격) 3종, 특수 몬스터(돌격, 관통 등) 2종
   
   ### ㄷ. 아이템 - 무기 3종, 패시브 3종

   ### ㄹ. 시스템 - 레벨 및 경험치 시스템, 타이머
   
   ### ㅁ. UI - 게임 시작화면, 하단 예시와 비슷하게 정보를 종합한 결과창, 그 외 이펙트

## 주차별 일정

1주차 - 발표 + 시작화면

2주차 - 플레이어 이동 및 맵 개발

3주차 - 기본 몬스터 개발

4주차 - 특수 몬스터 개발

5주차 - 무기 및 패시브 아이템 개발

6주차 - 무기 개발 및 세트 효과 제작

7주차 - 경험치 및 레벨 업 시스템

8주차 - 웨이브 시스템
